# General
- Windows and MacOS versions

# Distribution
- Windows version will use Windows installer
- MacOS version will be .dmg package

# SSH Functionality
- SSH capability
- Have ability to upload SSH keys
- Default to port 22, but allow for changing the port number, if necessary

# Aesthetics 
- lucida console will be the default font
- linux standard terminal colors will be the default colors
- default terminal type will be VT100
